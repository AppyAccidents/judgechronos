# Judge Chronos

**Judge Chronos** is an open-source, privacy-focused time-tracking application built exclusively for macOS. It helps users understand and organize their digital activity by extracting, categorizing, and visualizing app usage data, all while respecting user privacy and system integrity.

## Purpose
Judge Chronos empowers macOS users to:
- Gain insight into how they spend time on their computer
- Organize activities by project or category
- Automate time categorization with simple rules
- Export and visualize their productivity data

## Preliminary Features
- **Automatic Activity Extraction:** Reads and displays app usage events from the system database (knowledgeC.db)
- **User-Friendly Permissions:** Guides users through granting Full Disk Access with clear onboarding
- **Timeline View:** Chronological, non-editable list of daily app usage
- **Manual Categorization:** Assign activities to projects or categories
- **Dashboard & Reporting:** Visualize time spent per project/category with charts
- **Automation Rules:** Auto-categorize activities based on user-defined rules
- **Idle Detection:** Identify and handle idle periods
- **Export:** Export activity data to CSV

## Tech Stack
- Swift, SwiftUI
- SQLite.swift
- SwiftData
- SwiftUI Charts
